1. Create a virtual environmentconda create --name DistributedComputing python=3.7 -y2. Activating the environmentconda activate DistributedComputing3. Determining the current environment and list environmentconda info --envs conda env list5. Deactivate the environmentconda deactivate6. Activate againconda activate DistributedComputing7. Export the environment file to share and reproduce the environments including all the packages with corresponding versions.conda install numpyconda env export > DistributedComputing_environment.yml8. Remove the environmentconda deactivateconda remove --name DistributedComputing --all9.Create or update an environment using .yml.conda env create -f DistributedComputing_environment.yml -n DistributedComputing
